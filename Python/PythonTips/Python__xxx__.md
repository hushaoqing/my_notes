# Python 用下划线作为变量前缀和后缀指定特殊变量
_xxx 不能用’from module import *’导入
\__xxx__ 系统定义名字
__xxx 类中的私有变量名
核心风格：避免用下划线作为变量名的开始。
```
object.__class__
object.__delattr__
object.__doc__
object.__format__
object.__getattribute__
object.__hash__
object.__init__
object.__new__
object.__reduce__
object.__reduce_ex__
object.__repr__
object.__setattr__
object.__sizeof__
object.__str__
object.__subclasshook__
```
